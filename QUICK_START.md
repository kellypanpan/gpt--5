# 🚀 GPT-5AI 快速启动指南

## ⚡ 立即开始测试

你的GPT-5AI平台已经配置完成！现在可以立即测试Clerk认证和OpenAI API功能。

### 1. 启动开发服务器

```bash
npm run dev
```

### 2. 访问测试页面

打开浏览器访问：**http://localhost:5173/tools/writer**

### 3. 测试步骤

#### 步骤 1: 测试认证
- 点击右上角 "Sign In" 按钮
- 使用Clerk进行登录/注册
- 登录成功后应该看到 "✅ 已认证" 状态

#### 步骤 2: 测试OpenAI API连接
- 在页面中找到 "OpenAI API 连接测试" 卡片
- 点击 "测试 OpenAI API 连接" 按钮
- 应该看到 "✅ API 连接成功" 消息

#### 步骤 3: 测试文本生成
- API连接成功后，会显示 "步骤 2: 测试文本生成功能"
- 输入测试提示词（或使用默认的）
- 点击 "测试文本生成" 按钮
- 几秒后应该看到AI生成的内容

## 🔧 如果遇到问题

### 常见问题排查

1. **登录按钮无响应**
   ```bash
   # 检查Clerk配置
   echo $VITE_CLERK_PUBLISHABLE_KEY
   ```

2. **API连接失败**
   ```bash
   # 检查OpenAI密钥
   echo $OPENAI_API_KEY
   ```

3. **页面无法访问**
   ```bash
   # 确认开发服务器运行
   npm run dev
   ```

### 错误代码含义
- `401 Unauthorized` - 需要登录
- `402 Payment Required` - 积分不足
- `429 Too Many Requests` - 请求过于频繁
- `500 Internal Server Error` - 服务器错误
